# Purpose
Learning PostgreSQL

# Config
1. Copy database-config.env and use it as a template create a database.env file in root folder

# Connect
1. docker-compose up
2. psql -h localhost -p 5432 -d "POSTGRES_DB" -U "POSTGRES_USER"
